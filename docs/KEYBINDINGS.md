# PathTree Keybindings

Documents all keyboard shortcuts and their behavior in each user-interface context.

## Navigation & Core Actions (Main Screen Tree View)

- `Enter` / Double Left-Click: Activate selected node (trigger default resource action).
- `j` / `Down`: Move cursor down.
- `k` / `Up`: Move cursor up.
- `h` / `Left`: Collapse active directory node or go to parent if already collapsed.
- `l` / `Right`: Expand active directory node.
- `/` / `s`: Focus Search Input.
- `a`: Open Add Node Dialog.
- `e`: Open Edit Node Dialog.
- `m`: Open Move Node Dialog.
- `d` / `Delete`: Open Confirm Delete Dialog.
- `w`: Jump to the next visible Workspace node.
- `W` / `Shift+W`: Jump to the previous visible Workspace node.
- `f`: Jump to the next visible Folder node (within the current workspace scope).
- `F` / `Shift+F`: Jump to the previous visible Folder node (within the current workspace scope).
- `o` / `O` / `Shift+O`: Open Resource Action Menu overlay (context actions, including Pin/Unpin).
- `p` / `P` / `Shift+P`: Open Global Pinned Resources Screen.

## Global Pinned Resources Screen

- `Enter` / Left-Click: Activate the highlighted pin (locate and select the original node in the main tree view).
- `u` / `d` / `Delete`: Unpin the highlighted node.
- `[`: Move the highlighted pin up (decrement position index).
- `]`: Move the highlighted pin down (increment position index).
- `Escape`: Close Pins screen.
